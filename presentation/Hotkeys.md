# Hotkeys

| Hotkey                                           | Description                     |
| ------------------------------------------------ | ------------------------------- |
| <kbd>⇧</kbd><kbd>Esc</kbd>                       | Dismiss quick suggestions panel |
| <kbd>⌃</kbd><kbd>⌘</kbd><kbd>I</kbd>             | Chat sidebar                    |
| <kbd>⌘</kbd><kbd>I</kbd>                         | Open inline chat                |
| <kbd>⇧</kbd><kbd>⌥</kbd><kbd>⌘</kbd><kbd>L</kbd> | Open quick chat                 |
