# Backend
The backend of this project lives at `/backend` and uses django and django-rest-framework.
DRF serializers should be used for all outgoing API response formats.
# Frontend
The Frontend of this project lives at `/frontend` and primarily uses react, typescript, vite, and tailwind.
## General purpose components
- Place all general-purpose components under `src/components`.
- Components should live in a folder that is named the same as the component.
- Component files should be named the same as the component itself.
- Components should be the default export of their file.
- All project icons should use FontAwesome
